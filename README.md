# PalindroneCheckerApp
**Objective**
To create a console-based Java application that checks whether a given string is a **palindrome** and helps understand basic Java concepts like **classes, main method, and string handling**.

**Project Flow**

1. Program starts and `main()` method runs.
2. Application displays welcome message and details.
3. User enters a string.
4. Program checks if the string is a palindrome.
5. Result is shown on the console.

**Use Case (UC1)**
When the program starts, it displays the **application name and version** using `System.out.println()`.
